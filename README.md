## Installeren
1. Clone repo
    ```
    $ git clone https://github.com/mich97/browser-technologies-1920.git
    ```
2. Move to directory
    ```
    $ cd browser-technologies-1920
    ```
3. Install dependencies
    ```
    $ npm install
    ``` 
4. Run
    ```
    $ npm run start
    ```
5. Open following url in browser
    ```
   localhost:3000
   ```
